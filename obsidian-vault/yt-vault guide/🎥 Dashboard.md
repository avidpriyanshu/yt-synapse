# Video Discovery Dashboard

Search and browse all videos in your vault using the table below. Use the search box to find videos by title, channel, or topics.

## How to Use

- **Sort**: Click column headers to sort by date, channel, or title
- **Search**: Use the Bases search box (top) to filter videos by title, channel name, or topic
- **Topics**: Click on a topic in the **topics** column to navigate to that topic page
- **Creators**: Click on a channel name to navigate to the creator page (auto-generated in Phase 3)
