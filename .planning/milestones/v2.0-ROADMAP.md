## Key Decisions

- **Templates:** Non-breaking forward approach (new templates only for new clips)
- **Topic Extraction:** Lightweight regex-based approach (no ML overhead)
- **User Control:** Flat tag system for channel categories (hierarchies deferred)
- **Logging:** Plain English throughout (non-technical audience focus)
- **Service Model:** Obsidian-native UI (no separate web app)
